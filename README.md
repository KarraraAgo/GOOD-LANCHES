🍔 GOOD LANCHES

O GOOD LANCHES é um site simples e intuitivo desenvolvido para uma lanchonete, com o objetivo de apresentar o cardápio de forma rápida e facilitar o contato com os clientes. O projeto foi construído utilizando HTML e CSS, priorizando desempenho, organização e facilidade de uso.


---

📌 Objetivo

O sistema tem como principal finalidade permitir que usuários visualizem os produtos disponíveis e acessem rapidamente formas de contato para realizar pedidos, garantindo praticidade e agilidade na navegação.


---

⚙️ Requisitos Funcionais

O sistema apresenta as seguintes funcionalidades:

Exibição do cardápio com produtos, nomes e preços

Navegação simples entre as seções da página

Botões de ação para facilitar o pedido (ex: WhatsApp)

Exibição de informações da lanchonete (nome, contato e redes sociais)

Links externos para redes sociais e comunicação direta com o cliente



---

🛠️ Requisitos Não Funcionais

O projeto também atende aos seguintes critérios de qualidade:

Usabilidade: interface simples, intuitiva e de fácil navegação

Responsividade: adaptação a diferentes dispositivos (celular, tablet e desktop)

Desempenho: carregamento rápido por utilizar tecnologias leves

Compatibilidade: funcionamento nos principais navegadores modernos

Manutenibilidade: código organizado e separado (HTML e CSS)

Design: layout visual agradável com identidade moderna

Acessibilidade básica: boa legibilidade e contraste de cores



---

💻 Tecnologias Utilizadas

HTML5

CSS3



---

🚀 Considerações Finais

O GOOD LANCHES é um projeto focado na simplicidade e eficiência, sendo ideal para pequenos negócios que desejam ter presença online de forma prática, rápida e funcional.
